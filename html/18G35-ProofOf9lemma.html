proof of 9-lemma


As in the proof of the 5-lemma, we assume without loss of generality that we are working in modules over a ring. In keeping with the notion that the maps between the A’s (as well as between the B’s and the C’s) are cohomology sequences, we denote all vertical maps by d. The map Ai→Bi is denoted αi, and the map Bi→Ci is denoted βi. We must show that

  1. 1.

    β1 is surjective;

  2. 2.

    α1 is injective;

  3. 3.

    ker⁡β1⊂im⁢α1;

  4. 4.

    β1∘α1=0 (i.e. ker⁡β1⊃im⁢α1)

β1 is surjective: Choose c∈C1. Then d⁢c=β2⁢b, and β3⁢d⁢b=d⁢β2⁢b=d2⁢c=0, so d⁢b=α3⁢a=α3⁢d⁢a′. Thus d⁢(b-α2⁢a′)=0, so d⁢b′=b-α2⁢a′. Finally, d⁢β1⁢b′=β2⁢d⁢b′=β2⁢(b-α2⁢a′)=β2⁢b=d⁢c. But d is injective, so c=β1⁢b′.

α1 is injective: This is clear, since d⁢α1=α2⁢d, and α2 and both d’s are injective.

ker⁡β1⊂im⁢α1: Suppose β1⁢(b)=0. Then β2⁢d⁢b=d⁢β1⁢b=0, so d⁢b=α2⁢a. But then α3⁢d⁢a=d⁢α2⁢a=d2⁢b=0, and α3 is injective, so a∈ker⁡d and d⁢a′=a. Finally, d⁢α1⁢a′=α2⁢d⁢a′=α2⁢a=d⁢b. d is injective and thus b=α1⁢a′.

β1∘α1=0: d⁢β1⁢α1=β2⁢α2⁢d=0. But d is injective, so β1⁢α1=0.

Similar diagram chasing can be used to prove that if the top two rows are exact then so is the bottom row.
