simplicial object


A simplicial object in a category C is a contravariant functor from the simplicial category Δ to C. Such a functor X is uniquely specified by the morphisms X⁢(δin):X⁢([n])→X⁢([n-1]) and X⁢(σin):X⁢([n])→X⁢([n+1]), which satisfy

X⁢(δin-1)⁢X⁢(δjn) = X⁢(δj-1n-1)⁢X⁢(δin) for ⁢i<j, (1)
X⁢(σin+1)⁢X⁢(σjn) = X⁢(σj+1n+1)⁢X⁢(σin) for ⁢i≤j, (2)
X⁢(δin+1)⁢X⁢(σjn) = {X⁢(σj-1n-1)⁢X⁢(δin)if ⁢i<j,idnif ⁢i=j⁢ or ⁢i=j+1,X⁢(σjn-1)⁢X⁢(δi-1n)if ⁢i>j+1. (3)

In particular, a simplicial set is a simplicial object in 𝐒𝐞𝐭. Equivalently, one could say that a simplicial set is a presheaf on Δ. The object X⁢([n]) of a simplicial set is a set of n-simplices, and is called the n-skeleton.
