example of universe of finite sets


This is an example of a class of universes in which all sets are finite. Not only are they good for illustrating the notion of universe, but they are appropriate for formalizing finite math.

To construct our universe, we begin with a finite set 𝐔0 and apply an inductive process. We shall illustate the construction more explicitly for a particular choice of 𝐔0 later.

We define the sets 𝐔i for i>0 as follows: 𝐔i is the set of all finite sets each of whose elements lie in some 𝐔j with j<i. Then we define 𝐔 as

𝐔=⋃i=0∞𝐔i.

To show that 𝐔 is a universe, we need to check that it satisfies the four defining properties. This is easily done as follows:

  1. 1.

    If x∈𝐔, then x∈𝐔i for some i. By definition, if y∈x, then y∈𝐔j for some j such that j<i. Since 𝐔j⊂𝐔, we conclude that y∈𝐔.

  2. 2.

    If x,y∈𝐔, then x∈𝐔i and y∈𝐔j for some i,j. Pick k such that k>i and k>j. Then, by definition, {x,y}∈𝐔k. Since 𝐔k⊂𝐔, we conclude that {x,y}∈𝐔.

  3. 3.

    If x∈𝐔, then x∈𝐔i for some i. Hence, every element of x belongs to some 𝐔j with j<i. Also, by definition, x is finite. Hence, every subset of x is a finite set every element of which lies belongs to some 𝐔j with j<i. This means that every subset of x belongs to 𝐔i. Since there are only a finite number of subsets of x and each of these is an element of 𝐔i, it follows that 𝒫⁢(x)∈𝐔i+1⊂𝐔.

  4. 4.

    Let us first show that, if x,y∈𝐔, then x∪y∈𝐔. By definition, there exist i,j such that x∈𝐔i and y∈𝐔j for some i,j. Pick k such that k>i and k>j. Since every element of x lies in 𝐔m for some m<i and every element of y lies in 𝐔n for some n<j and n,m<j, it follows that every element of x∪y belongs to 𝐔l for some l<k. Hence, x∪y∈𝐔k⊂𝐔. More generally, this implies that any finite union of elements of 𝐔 must lie in 𝐔. Since every element of 𝐔 is finite, every family {xi|i∈I∈𝐔} is finite, hence ∪i∈Ixi∈𝐔.
