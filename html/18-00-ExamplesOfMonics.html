examples of monics


This entry lists some common examples of monics (monomorphisms). The examples also illustrate some of the techniques in finding monics.

  1. 1.

    In Set, the category of sets, the monics are exactly the one-to-one functions. First, suppose f:A→B is one-to-one and that g,h:C→A are functions such that f∘g=f∘h. Then for any c∈C, f⁢(g⁢(c))=f⁢(h⁢(c)). Since f is 1-1, g⁢(c)=h⁢(c), or g=h, or f is monic. Conversely, suppose f:A→B is monic and that f⁢(x)=f⁢(y). Define functions g,h:{z}→A by g⁢(z)=x and h⁢(z)=y. Then f∘g⁢(z)=f∘h⁢(z) by assumption. But this means that g⁢(z)=h⁢(z) since f is monic. Hence x=y, or f is 1-1.

  2. 2.

    In Grp, the category of groups, the monics are exactly the injective group homomorphisms. If f:G→H is 1-1 and f∘g=f∘h for group homomorphisms g,h:K→G. Then for any k∈K, f⁢(g⁢(k))=f⁢(h⁢(k)), so that g⁢(k)=h⁢(k). Hence f is monic. Conversely, suppose that f is monic and f⁢(x)=f⁢(y). Let g,h:ℤ→G be group homomorphisms given by g⁢(1)=x and h⁢(1)=y. Then f∘g⁢(1)=f⁢(x)=f⁢(y)=f∘h⁢(1). This means that x=g⁢(1)=h⁢(1)=y, or f is 1-1.

  3. 3.

    In Rng, the category of rings (with 1), the monics are exactly the injective ring homomorphisms. If f:R→S is a 1-1 ring homomorphism, and f∘g=f∘h for some ring homomorphisms g,h:T→R, then for any t∈T, f(g(t)=f(h(t)), so that g⁢(t)=h⁢(t), or g=h, so that f is monic. Conversely, let f be monic and f⁢(x)=f⁢(y). Define g,h:ℤ⁢[X]→R be given by g⁢(X)=x and h⁢(X)=y. Once again, we see that f⁢(g⁢(X))=f⁢(x)=f⁢(y)=f⁢(h⁢(X)), so that x=g⁢(X)=h⁢(X)=y since f is monic. Therefore, f is 1-1.

In DivAbGrp, the category of divisible abelian groups, every injection is a monic, but not conversely. If f:A→B is 1-1 and f∘g=f∘h for g,h:C→A, then f∘(g-h)=0, so that 0=(g-h)⁢(x)=g⁢(x)-h⁢(x) for all x∈C. This means that g=h. The counterexample comes from the following: let f:ℚ→ℚ/ℤ be the canonical projection. It is clear that f is not 1-1. Yet f is a monomorphism: note first that f∘g=f∘h is equivalent to f∘(g-h)=0, so we may as well assume that f∘g=0 and show that g=0 for any g:C→ℚ. If not, then for some a∈C, 0≠g⁢(a)=p/q∈ℚ, where p is a positive integer and q∈ℤ. If q=1, set n=2⁢p; otherwise, set n=p. Then f∘g⁢(a/n)=f⁢(1/2)=1/2 if q=1, and f∘g⁢(a/n)=f⁢(1/q)=1/q otherwise. In either case, f∘g⁢(a/n)≠0, a contradiction.

As another example where monomorphisms are not necessarily injective, it can be shown that in the category of pointed, connected topological spaces (with continuous functions preserving base points), the projection of the circular helix onto the unit circle (clearly not injective) is a monomorphism.
