constant functor


Let 𝒞 and 𝒟 be categories. A constant functor from 𝒞 to 𝒟 is a functor k:𝒞→𝒟 such that there is an object A∈𝒟 such that

  • •

    for all objects X in 𝒞, k⁢(X)=A, and

  • •

    for all morphisms X→Y in 𝒞, k(X→Y)=1A, the identity morphism of A.

To see that this is indeed a functor, we merely need to verify that

k((X→Y)∘(Y→Z))=k(X→Y)∘k(Y→Z).

But this is obvious, as the left hand side is k(X→Z)=1A, while the right hand side is 1A∘1A=1A.

Remarks.

  • •

    For the constant functor k considered above, the object A is the fixed value of k. To identify k with A, we often write kA:=k, or simply A when no confusion arises.

  • •

    Composing a functor with a constant functor gives us a constant functor. More precisely, let kA:𝒞→𝒟 be the constant functor with fixed value A. If F:ℬ→𝒞 is a functor, then kA∘F:ℬ→𝒟 is the constant functor (k∘F)A with fixed value at A. Moreover, if G:𝒟→ℰ is a functor, then G∘kA:𝒞→ℰ is the constant functor (G∘k)G⁢(A) valued at G⁢(A).

  • •

    Given any functor T:𝒞→𝒟, any natural transformation τA:kA⁢→˙⁢T takes any object X∈𝒞 to a morphism A→T⁢(X), and, for any morphism α:X→Y in 𝒞, a commutative triangle

    \xymatrix⁢@⁢R=7⁢p⁢t⁢@⁢C=1.5⁢c⁢m⁢&⁢T⁢(X)⁢\ar⁢[d⁢d]T⁢(α)⁢A⁢\ar⁢[d⁢r]⁢\ar⁢[u⁢r]⁢&⁢&⁢T⁢(Y)

    in 𝒟. Similarly, any natural transformation σA:T⁢→˙⁢kA takes any object X to T⁢(X)→A and any morphism α:X→Y to a commutative triangle

    \xymatrix⁢@⁢R=7⁢p⁢t⁢@⁢C=1.5⁢c⁢m⁢T⁢(X)⁢\ar⁢[d⁢d]T⁢(α)⁢\ar⁢[d⁢r]⁢&⁢&⁢A⁢T⁢(Y)⁢\ar⁢[u⁢r]⁢&
  • •

    If α:A→B is any morphism in 𝒟, then the natural transformation τ:kA⁢→˙⁢kB given by sending every object X∈𝒞 to the morphism τX:A→B:=α can be thought of as a “constant” natural transformation, since, for any morphism β:X→Y in 𝒞, the commutative diagram

    \xymatrix@R=1.25cm@C=2cmkA(X)\ar[d]kA⁢β\ar[r]τX&\ar@[dr]|=kB(X)\ar[d]kB⁢β&A\ar[d]1A\ar[r]α&B\ar[d]1BkA(Y)\ar[r]τY&kB(Y)&A\ar[r]α&B

    reduces to α.

  • •

    As above, denote τ by τα. Let σA:T⁢→˙⁢kA be a natural transformation. Then the transformation τα∘σA:T⁢→˙⁢kB sends any object X in 𝒞 to the morphism

    T⁢(X)→A→αB

    in 𝒟, and any morphism β:X→Y to the commutative diagram

    \xymatrix⁢@⁢R=7⁢p⁢t⁢@⁢C=1.5⁢c⁢m⁢T⁢(X)⁢\ar⁢[d⁢d]T⁢(β)⁢\ar⁢[d⁢r]⁢&⁢&⁢&⁢A⁢\ar⁢[r]α⁢&⁢B⁢T⁢(Y)⁢\ar⁢[u⁢r]⁢&⁢&

    For any natural transformation γB:kB⁢→˙⁢T, the composition σB∘τα works similarly.
