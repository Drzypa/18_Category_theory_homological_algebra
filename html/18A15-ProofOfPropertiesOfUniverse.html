  4. 4.

    By axiom 2, {x,y}∈𝐔. If we set zx=x and zy=y, then x∪y=⋃i∈{x,y}zi, hence, by axiom 4, x∪y∈𝐔. — If x∈U and y∈U then, by axiom 1, a∈𝐔 for all a∈x and b∈𝐔 for all b∈y. By property 3, if a∈𝐔 and b∈𝐔, then (a,b)∈𝐔; further, by property 1, {(a,b)}∈𝐔. Hence, by axiom 4, {(a,b)∣b∈y}=⋃b∈y{(a,b)}∈𝐔 for all a∈x.   5. 5.

    By axiom 4, ⋃i∈Ixi∈𝐔. By property 4, I×⋃i∈Ixi∈𝐔. Now, every function from I to ⋃i∈Ixi∈𝐔 is a subset of I×⋃i∈Ixi∈𝐔. Since ∏i∈Ixi is a set of functions from I to ⋃i∈Ixi∈𝐔, we have, by defintion of power set, ∏i∈Ixi⊂(P)⁢(I×⋃i∈Ixi). Hence, by axiom 3 and property 2, we conclude that ∏i∈Ixi∈𝐔.

    Assume the contrary, namely that x∈𝐔 and #⁢x≥#⁢𝐔. By axiom 3, 𝒫⁢(x)∈𝐔 but #⁢(𝒫⁢(x))=2#⁢x≥2#⁢𝐔.
