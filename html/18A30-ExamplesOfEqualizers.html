examples of equalizers


This entry illustrates some common examples of equalizers and coequalizers.

Examples of Equalizers

  • •

    In Set, the category of sets, the equalizer of a pair of functions f,g:A→B is given by the following: a set

    C={x∈A∣f⁢(x)=g⁢(x)},

    and function i:C→A the canonical injection. Clearly i equalizes f and g by construction: f∘i=g∘i. Now, if j:D→A also equalizes f and g, then define k:D→C by k⁢(d)=j⁢(d). To see that this is well-defined, we need to show that j⁢(d)∈C. Since j equalizes f and g, we have f⁢(j⁢(d))=g⁢(j⁢(d)), so that j⁢(d)∈C. Therefore k is a well-defined function from D into C. In addition, i∘k⁢(d)=i⁢(j⁢(d))=j⁢(d). Finally, it is easy to see that if i∘t=j, then t=k. Therefore, (C,i) is the equalizer of f and g.

  • •

    In fact, most concrete categories (concrete over Sets), the equalizer of a pair of morphisms is given by the object C above with i the corresponding injective mapping.

Examples of Coequalizers

  • •

    In Set, the coequalizer of a pair of functions f,g:A→B can be found as follows: define a binary relation ∼ on B such that for any x,y∈B, x∼y iff either x=y, or there is an a∈A such that h1⁢(a)=x and h2⁢(a)=y, where h1,h2∈{f,g}. Then ∼ is easily seen to be a reflexive symmetric relation. Now, take the transitive closure ∼* of ∼. So ∼* is an equivalence relation on B. Let B/∼* be the set of all the equivalence classes, and p:B→B/∼* the canonical projection. Then (B/∼*,p) is the coequalizer of f and g. First, p∘f⁢(a)=[f⁢(a)]=[g⁢(a)]=p∘g⁢(a), since f⁢(a)∼g⁢(a). Suppose now that q:B→D is another function that coequalizes f and g. Define r:B/∼*→D by r⁢([b])=q⁢(b). We want to show that r is well-defined. In other words, if b∼*c, then q⁢(b)=q⁢(c). First, assume b∼c. Then either b=c (in which case, q⁢(b)=q⁢(c) is immediate), or there is a∈A such that h1⁢(a)=b and h2⁢(a)=c, with h1,h2∈{f,g}. In this case, q⁢(b)=q⁢(h1⁢(a))=q⁢(h2⁢(a))=q⁢(c) since q∘f=q∘g. Now, if b∼*c, then there are d1,…,dn∈B such that b=d1∼d2∼⋯∼dn=c. As a result, q⁢(b)=q⁢(d1)=q⁢(d2)=⋯=q⁢(dn)=q⁢(c). In addition, r∘p⁢(b)=r⁢([b])=q⁢(b), and that r is uniquely determined this way. Therefore, (B/∼*,p) is the coequalizer of f and g.
