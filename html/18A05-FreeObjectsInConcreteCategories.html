free objects in concrete categories


By concrete category we will understand pair (𝒞,U), where C is a category and U:𝒞→𝒮⁢ℰ⁢𝒯 is a faithful (covariant) functor. Assume that (𝒞,U) is a concrete category.

Definition 1. Let X be an object in 𝒞. Subset B⊆U⁢(X) (possibly empty) is called a basis of X if for any object Y in 𝒞 and any function g:B→U⁢(Y) there exists exactly one morphism α:X→Y such that U⁢(α)⁢(x)=g⁢(x) for any x∈B. In this case we will say that g lifts to α.

Definition 2. Object X will be called free if there exists basis of X.

Free objects generalize the notion of free modules over a ring. Some of the properties of free modules can be easily generalized to free objects in arbitrary concrete category. For example:

Proposition. Let X and Y be free objects with bases B and B′ respectively and let f:B→B′ be a function. The following statements hold:
i) If f is an injection, then there exists a section α:X→Y in 𝒞;
ii) If f is a surjection, then there exists a retraction β:X→Y in 𝒞;
iii) If f is a bijection, then X and Y are isomorphic.

Proof. i) Assume that f:B→B′ is an injection. Let f1:B→U⁢(Y) be defined as

f1⁢(x)=f⁢(x)

for all x∈B. Now, since f:B→B′ is an injection, then there exists a surjection f′:B′→B such that

f′⁢(f⁢(x))=x

for all x∈B. Let f2:B′→U⁢(X) be defined by

f2⁢(y)=f′⁢(y)

for all y∈B′. Now both X and Y are free and thus there are morphism α:X→Y and β:Y→X such that

U⁢(α)⁢(x)=f1⁢(x)⁢ and ⁢U⁢(β)⁢(y)=f2⁢(y)

for all x∈B and y∈B′. It is easy to check, that this implies that

U⁢(β∘α)⁢(x)=x

for all x∈B. But U⁢(idX)⁢(x)=x for all x∈B and thus canonical injection i:B→U⁢(X) lifts to both β∘α and idX. Since lift is unique, then β∘α=idX, so α is a section.

ii) Note that if f:B→B′ is a surjection, then there exists an injection g:B′→B such that f⁢(g⁢(y))=y for all y∈B′. Thus, from i) we obtain that β∘α=idY for α:Y→X and β:X→Y constructed as in i). Therefore β:X→Y is a retraction.

iii) If f is a bijection, then proof of i) and ii) shows that there are two morphisms α:X→Y and β:Y→X such that β∘α=idX and α∘β=idY. Thus X and Y are isomorphic. □

Remark 1. Free objects does not have to exist. For example, the category of finite groups (without the trivial group) and group homomorphisms (where U is a forgetful functor) does not have free objects (this is because there are no nontrivial group homomorphisms between groups with relatively prime orders).

Remark 2. Note that, if there is a free object X in a concrete category (𝒞,U) such that ∅ is a basis of X, then X is an initial object. This follows directly from the definition, since any morphism α:X→Y is a lift of f:∅→U⁢(Y), thus it has to be unique. Conversly one can easily show, that initial object is always free with ∅ as a basis.
