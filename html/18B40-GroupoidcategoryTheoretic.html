groupoid (category theoretic)


A groupoid, also known as a virtual group, is a small category where every morphism is invertible. We can give a more explicit, algebraic definition: start with a set G, and a partial binary operation ∘ on G. Call a pair (x,y) of elements of G a composable pair if (x,y)∈dom⁡(∘). A groupoid is the pair (G,∘), together with two unary operations eL and eR on it, satisfying the following conditions:

  1. 1.

    (x,y) is a composable pair iff eR⁢(x)=eL⁢(y).

  2. 2.

    (x,y) and (x∘y,z) are composable pairs iff (y,z) and (x,y∘z) are, and if one of these is true, then (x∘y)∘z=x∘(y∘z).

  3. 3.

    (eL⁢(x),x) and (x,eR⁢(x)) are composable pairs and x=eL⁢(x)∘x=x∘eR⁢(x).

  4. 4.

    for each x∈G, there exists y∈G such that (x,y) and (y,x) are composable pairs, and eL⁢(x)=x∘y and eR⁢(x)=y∘x.

Below are some properties:

  1. 1.

    In condition 4 above, eL⁢(x)=eR⁢(y) and eR⁢(x)=eL⁢(y). This is true by condition 1, since both (x,y) and (y,x) are composable pairs.

  2. 2.

    Again, in condition 4, y is unique. To see this, suppose z∈G satisfies condition 4 (in place of y). Then y=y∘eR⁢(y)=y∘eL⁢(x)=y∘(x∘y)=y∘(x∘z)=(y∘x)∘z=(z∘x)∘z=eR⁢(x)∘z=eL⁢(z)∘z=z. Notice property 1 is used in the proof. We call y the inverse of x, and write x-1.

  3. 3.

    In view of condition 4, both eL and eR are unique. In other words, if fL,fR:G→G are unary operators on G satisfying conditions 3 and 4 above (in place of eL and eR), then fL=eL and fR=eR. In fact, eL⁢(x)=x∘x-1 and eR⁢(x)=x-1∘x.

  4. 4.

    Since x=eL⁢(x)∘x=eL⁢(x)∘(eL⁢(x)∘x)=(eL⁢(x)∘eL⁢(x))∘x, we see that eL⁢(x) is composable with itself, and that eL⁢(x)∘eL⁢(x)=eL⁢(x) by the previous property. Similarly, eR⁢(x)∘eR⁢(x)=eR⁢(x). This shows that eR⁢(x) and eL⁢(x) are idempotent with respect to ∘ for every x∈G.

  5. 5.

    Since (eL⁢(x),x) is a composable pair, eR⁢(eL⁢(x))=eL⁢(x) for any x∈G. Similarly, eL⁢(eR⁢(x))=eR⁢(x). Hence eR⁢(eR⁢(x))=eR⁢(eL⁢(eR⁢(x)))=eL⁢(eR⁢(x))=eR⁢(x). Similarly, eL⁢(eL⁢(x))=eL⁢(x). This shows that eR and eL are idempotent with respect to functional compositions.

  6. 6.

    (Cancellation property): if x∘y=x∘z, then y=z; if y∘x=z∘x, then y=z.

    Proof.

    Since (x,y) is a composable pair, eR⁢(x)=eL⁢(y). But eR⁢(eR⁢(x))=eR⁢(x), we have eR⁢(eR⁢(x))=eL⁢(y) so that (eR⁢(x),y)=(x-1∘x,y) is a composable pair, hence (x-1,x∘y) is a composable pair and x-1∘(x∘y)=(x-1∘x)∘y=eR⁢(x)∘y. Since (eR⁢(x),y) is a composable pair, eR⁢(x)=eR⁢(eR⁢(x))=eL⁢(y). As a result, x-1∘(x∘y)=eL⁢(y)∘y=y. Similarly x-1∘(x∘z)=z. By assumption, we deduce that y=z. The other statement is proved similarly. ∎

  7. 7.

    The algebraic definition given can be easily turned into a categorical definition (using objects and morphisms). The details are left for the reader.

If eR and eL are constant functions, then G is a group.

Remark. There is also a group-theoretic concept (http://planetmath.org/Groupoid) with the same name.
