example of exact functor


In this entry, we show the following proposition:

Proposition 1.

hom⁡(F,-) is an exact functor for any free module F over a ring R.

Before proving this, let us prove something related to a free module:

Lemma 1.

If f:F→N is a module homomorphism where F is free, then for any module homomorphism α:M→N where im⁡(f)⊆im⁡(α), there is a module homomorphism g:F→M such that α∘g=f.

Proof.

The case when F=0 is trivial. Suppose now F≠0 is free, F has a basis, say X. For each x∈X, let Mx:=α-1⁢(f⁢(x)). Each Mx is non-empty. By the axiom of choice (one of its equivalents if necessary), there is a function g from X to ⋃{Mx∣x∈F}⊆M such that g⁢(x)∈Mx. Since X is a basis for F, this function can be extended to a module homomorphism from F to M. By abuse of notation, let us use g for the extension. Since α⁢(g⁢(x))=f⁢(x) for all x∈X, this implies that α∘g=f on all of F. ∎

Proof of Proposition 1.

Let

\xymatrix⁢0⁢\ar⁢[r]⁢&⁢A⁢\ar⁢[r]α⁢&⁢B⁢\ar⁢[r]β⁢&⁢C⁢\ar⁢[r]⁢&⁢0

be a short exact sequence of R-modules. We want to show that

\xymatrix⁢0⁢\ar⁢[r]⁢&⁢hom⁡(F,A)⁢\ar⁢[r]α*⁢&⁢hom⁡(F,B)⁢\ar⁢[r]β*⁢&⁢hom⁡(F,C)⁢\ar⁢[r]⁢&⁢0

is short exact. This amounts to establishing the following three equations:

  • •

    ker⁡(α*)=0:

    If α*⁢(f)=0, then α∘f=0, which implies that f⁢(x)=0 for all x∈F. This means that f=0.

  • •

    im⁡(β*)=hom⁡(F,C):

    By Lemma 1 above, for every f:F→C, there is a g:F→B such that β∘g=f, or β*⁢(g)=f.

  • •

    im⁡(α*)=ker⁡(β*):

    If f∈im⁡(α*), then there is g:F→A such that α∘g=f. Therefore, β*⁢(f)=β∘f=β∘(α∘g)=(β∘α)∘g=0∘g=0, showing that im⁡(α*)⊆ker⁡(β*).

    On the other hand, pick f∈ker⁡(β*), so f⁢(F)⊆ker⁡(β)=im⁡(α). Therefore, by Lemma 1 above, there is a g:F→A such that α∘g=f. This means that f∈im⁡(α*).

Thus, hom⁡(F,-) is exact. ∎

Remark. The converse of this is not true. But we do have the following fact: hom⁡(P,-) is an exact functor iff P is a projective module (over some ring R).
