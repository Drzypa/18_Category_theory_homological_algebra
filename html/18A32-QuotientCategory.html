quotient category


Quotient categories are a generalization of quotient structures found in many mathematical systems, such as sets and groups. For example, if we have a set S={a,b,c}, by identifying a and c as being the “same”, we obtain a two-element set {{a,c},b}, called the quotient set of A by identifying a and b. Here is a more useful example: take the set S of integers. Identify integers m and n if their difference is divisible by 2. Then we obtain a quotient of S consisting of two elements: the set of odd numbers and the set of even numbers.

In both of the examples above, we start with a set S and a binary relation R (in the first example, R is the singleton {(a,b)}, and in the second, R consists of pairs of integers (m,m+2⁢n), where m,n are integers) on S. The quotient Q of S by R then satisfies the following functional properties:

  1. 1.

    there is an onto map p:S→Q such that a⁢R⁢b implies p⁢(a)=p⁢(b), and

  2. 2.

    if we have another onto map q:S→T satisfying 1 above, then there is a unique map f:Q→T such that f∘p=q.

The two properties above are exactly what we need to define the quotient of a category 𝒞 by a binary relation R on 𝒞. But what is R precisely? We define a binary relation on a category 𝒞 to be a binary relation R on Mor⁡(𝒞), the class of morphisms of 𝒞. In other words, R is a class consisting of pairs (f,g) where f and g are morphisms in 𝒞. If (f,g) is in R, we write f⁢R⁢g.

Definition. Let 𝒞 be a category and R a binary relation on 𝒞. Then a quotient of 𝒞 by R is a pair (𝒬,P) where 𝒬 is a category and P:𝒞→𝒬 is a functor, satisfying the following two conditions:

  1. 1.

    P is a surjection on objects; that is, for every object Q in 𝒬, there is a object C in 𝒞, such that P⁢(C)=Q.

  2. 2.

    if f⁢R⁢g, then P⁢(f)=P⁢(g),

  3. 3.

    if (𝒟,F) is another pair satisfying conditions 1 and 2 above, then there is a unique functor G:𝒬→𝒟 such that G∘P=F.

It is easy to see that 𝒬 is unique up to natural isomorphism, if it exists. The category 𝒬 is called the quotient category of 𝒞 by R, written 𝒞/R.

To see how 𝒞/R may possibly be constructed, we make the following few observations:

  • •

    First, if we take the reflexive, symmetric, and transitive closure R* of R (so that R* is the smallest equivalence relation containing R), then f⁢R*⁢g implies P⁢(f)=P⁢(g).

  • •

    In addition, if f1⁢R*⁢g1 and f2⁢R*⁢g2 such that f1∘f2 and g1∘g2 are both defined, then P⁢(f1∘f2)=P⁢(g1∘g2). This means that we may reduce R* to a congruence relation R′ with respect to morphism composition. Once again, f⁢R′⁢g implies P⁢(f)=P⁢(g).

  • •

    Furthermore, if f:A→B and g:C→D and f⁢R′⁢g, then P⁢(f)=P⁢(g) forces P⁢(A)=P⁢(C) and P⁢(B)=P⁢(D). In other words, the domains of f and g are identified under P. The same is true for the corresponding codomains. This implies that, if [f] denotes the equivalence class of f under R′, then the domains (codomains) of all g∈[f] are identified as one object in 𝒞/R, if it exists.

  • •

    As a result, all the corresponding identity morphisms are identified as well, for if P⁢(A)=P⁢(B), then P⁢(1A)=1P⁢(A)=1P⁢(B)=P⁢(1B). This means we can further reduce R′ to R′′ by such identifications. Note that R′′ is still a congruence relation.

Note, however, the construction above does not guarantee that 𝒞/R in fact exists. This is mainly due to the identification of objects in 𝒞, which may result in blowing up some hom sets into proper classes. However, if

  1. 1.

    R is a set, or

  2. 2.

    if f⁢R⁢g implies that f and g are parallel (they have the same domain and codomain),

then 𝒞/R exists. In the first case, every equivalence class with respect to R′′ is a set, so that hom sets remain sets. In the second, R′′ is not even needed, so that no two objects are identified, and there is no danger in hom sets being blown up into classes. In fact, the second condition is how [1] defines a quotient category. Note that in this case, P is a bijection on objects between 𝒞 and 𝒞/R.

In the following discussion, we denote objects and morphisms in 𝒞/R by [A]R and [f]R (images of A and f under P), or simply [A] and [f] when there is no confusion.

Suppose 𝒞 is a category with a binary relation R on its morphisms. Below are some examples, as well as a non-example:

  1. 1.

    Suppose 𝒞 is generated by four objects A,B,C,D and three morphisms f:A→B, g:C→D, and h:B→C. Furthermore, suppose R={(f,g)}. Then 𝒞/R exists and is generated by objects X=[A]=[C] and Y=[B]=[D], and morphisms α=[f]:X→Y and β=[h]:Y→X. Note that whereas 𝒞 has only a finite number of morphisms (f,g,h and four identities), the quotient 𝒞/R has infinitely many. For example, on X, the morphisms are (β⁢α)n, for any non-negative integer n.

  2. 2.

    If R is defined so that f⁢R⁢g iff f is parallel to g, then R is easily seen to be a congruence relation. Then 𝒞/R is the category whose objects are objects of 𝒞, such that each hom set has at most one element. As a result, if both hom⁡(A,B) and hom⁡(B,A) are non-empty, then A is isomorphic to B.

  3. 3.

    The two examples above are artificial, here’s a concrete one: let 𝒞 be the category of topological spaces and continuous maps, define R so that f⁢R⁢g whenever f is homotopic to g. Then 𝒞/R is the category of topological spaces with homotopic classes of continuous maps.

  4. 4.

    Here’s an example where 𝒞/R does not exist. Let 𝒞 be a category whose collection of objects is a proper class, and there is a non-identity morphism hA on each object A. Define R so that f⁢R⁢g iff there are objects A,B with f=1A and g=1B. Then if 𝒞/R were to exist, it would have only one object, say X. Furthermore, hom⁡(X,X) contains a morphism [hA] for every A in 𝒞. Since [hA]=[hB] iff A=B, hom⁡(X,X) is not a set. Therefore, 𝒞/R is not well-defined.
