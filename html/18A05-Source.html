In the whole entry we suppose we are given a category 𝐀. By an object we always mean an object in 𝐀 and by a morphisms an 𝐀-morphism.

Definition 1.

A source in a category A is a pair (A,(fi)i∈I) where A is an object and fi:A→Ai are morphisms indexed by a class I.

The object A is called the domain of the source and the family (Ai)i∈I is called the codomain of the source.

A sink is a pair ((fi)i∈I,A) where A is an object and fi:Ai→A are morphisms.

Sources can be composed with morphisms. If 𝒮=(A,(fi)i∈I is a source and f:B→A is a morphism, we use the notation (B,(fi∘f)i∈I)=𝒮∘f. Similarly, for sinks, we use the notation f∘𝒮=((f∘fi)i∈I,B) if 𝒮=((fi)i∈I,A) is a sink and f:A→B is a morphism.

Definition 2.

A source S=(A,(fi)i∈I) is called a monosource if for any pair r,s:B→A of morphisms from the equality S∘r=S∘s follows r=s.

A sink S=((fi)i∈I,A) is called an episink if for any pair r,s:A→B of morphisms r=s whenever r∘S=s∘S.

A monosource S is called extremal monosource, if the following holds: Whenever S=S¯∘e for an epimorphism e, then e is an isomorphism.

An episink S is called extremal episink if the following holds: Whenever S=m∘S¯ pre for a monomorphism m, tak m is an isomorphism.

Every limit is an extremal monosource, a colimit is an extremal episink.
