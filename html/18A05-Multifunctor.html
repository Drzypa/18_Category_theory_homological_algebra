multifunctor


Let 𝒞 be the product of categories 𝒞1,…,𝒞n and 𝒟 be any category. A multifunctor F:𝒞→𝒟 satisfies the following

  • •

    for every A∈Ob⁡(𝒞), F⁢(A)∈Ob⁡(𝒟),

  • •

    for every α∈Mor⁡(𝒞), F⁢(α)∈Mor⁡(𝒟):F⁢(A)→F⁢(B), for some objects A,B∈Ob⁡(𝒞),

  • •

    There is a function ϕ:{1,…,n}→{id,op} where id is the identity functor and op is the opposite functor, with

    C¯:=ϕ⁢(1)⁢(𝒞1)×⋯×ϕ⁢(n)⁢(𝒞n)  and  αi¯:=ϕ⁢(i)⁢(αi)

    for any morphisms αi in 𝒞i, such that F¯:𝒞¯→𝒟 given by

    F¯⁢(A):=F⁢(A)  and  F¯⁢(α1,…,αn):=F⁢(α1¯,…,αn¯)

    is a covariant functor.

Condition 3 says that, while F may not be a functor, by appropriately changing some of the categories 𝒞i to their opposites, the newly defined F¯ becomes a functor. When there is no danger, we may identify F¯ with F.

Remarks.

  1. 1.

    The function ϕ in condition 3 above can be changed so that F¯ is a contravariant functor instead.

  2. 2.

    Since F¯ is a functor, by restricting F¯ to any coordinate gives us a functor as well. Formally, given object A=(A1,…,An)∈𝒞¯, if we define FA:ϕ⁢(i)⁢(𝒞i)→D by setting

    FA⁢(Bi)=F¯⁢(Ai^⁢(Bi))  and  FA⁢(βi)=F¯⁢(Ai^⁢(βi)),

    for each i, where

    1. (a)

      Ai^⁢(Bi) is the object in 𝒞¯ whose i-th coordinate is Bi and agrees with A everywhere else, and

    2. (b)

      Ai^⁢(βi) is the morphism in 𝒞¯ whose i-th coordinate is βi, and the identity morphism (on Aj) everywhere else,

    then FA is a covariant functor.

  3. 3.

    Furthermore, since F¯ is covariant, this means for any α=(α1,…,αn):A→B, we have the decomposition

    F¯⁢(α)=F¯⁢(α1^)∘⋯∘F¯⁢(αn^)

    where αi^:=Ai^⁢(αi) as defined in property 2 above.

  4. 4.

    In addition, we see that F¯⁢(αi^)∘F¯⁢(αj^)=F¯⁢(αj^)∘F¯⁢(αi^) for i≠j.

  5. 5.

    In fact, properties 2, 3, and 4 are enough to insure that F¯ is a covariant functor, for

    F¯⁢(α∘β) = F¯⁢((α1,…,αn)∘(αn,…,βn))
    = F¯⁢(α1∘β1,…,αn∘βn)
    = F¯⁢(α1∘β1^)∘⋯∘F¯⁢(αn∘βn^)
    = FA⁢(α1∘β1)∘⋯∘FA⁢(αn∘βn)
    = (FA⁢(α1)∘FA⁢(β1))∘⋯∘(FA⁢(αn)∘FA⁢(βn))
    = (F¯⁢(α1^)∘F¯⁢(β1^))∘⋯∘(F¯⁢(αn^)∘F¯⁢(βn^))
    = (F¯⁢(β1^)∘F¯⁢(α1^))∘⋯∘(F¯⁢(βn^)∘F¯⁢(αn^))
    = F¯⁢(β1∘α1^)∘⋯∘F¯⁢(βn∘αn^)
    = F¯⁢(β1∘α1,…,βn∘αn)
    = F¯⁢(β∘α)

    This means we can replace the statement that F¯ is a covariant functor in condition 3 of the definition by the three properties above.

  6. 6.

    F is called a bifunctor or trifunctor whenever n=2 or 3.

Hom functors. The most famous bifunctor is the hom functor from 𝒞×𝒞→𝐒𝐞𝐭. Given objects A,B in 𝒞, hom⁡(A,B) is the set of all morphisms from A to B. In addition, given morphisms α:A→B and β:X→Y, hom⁡(α,β) is the morphism from hom⁡(B,X) hom⁡(A,Y) taking f:B→X to g:=β∘f∘α:A→Y:

\xymatrix⁢B⁢\ar⁢[d]f⁢&⁢A⁢\ar⁢[l]α⁢\ar⁢[d]g⁢X⁢\ar⁢[r]β⁢&⁢Y

Let us verify that hom is indeed a “binary” multifunctor. Given any object A, we see that hom⁡(A,-) is covariant functor, for

hom⁡(1A,β∘α)⁢(f) = (β∘α)∘f∘1A=β∘(α∘f∘1A)
= β∘hom⁡(1A,α)⁢(f)=β∘hom⁡(1A,α)⁢(f)∘1A
= hom⁡(1A,β)⁢(hom⁡(1A,α)⁢(f))=hom⁡(1A,β)∘hom⁡(1A,α)⁢(f).

By the same reasoning, we see that, on the other hand, hom⁡(-,B) is contravariant for any object B. So we want to show that hom¯:𝒞op×𝒞→𝐒𝐞𝐭 is a covariant functor. Having just verified property 2 (see remarks above), we are left with properties 3 and 4. As the equation g=hom⁡(α,β)⁢(f) turns into g=hom¯⁢(α*,β)⁢(f), the diagram above turns into the commutative diagram below

\xymatrix⁢B⁢\ar⁢[d]f⁢\ar⁢[r]α*⁢&⁢A⁢\ar⁢[d]g⁢X⁢\ar⁢[r]β⁢&⁢Y

where α*:B→A is the opposite arrow of α. Now, properties 3 and 4 can be easily verified by the following commutative diagrams:

\xymatrixB\ar[d]f\ar[r]1B*&B\ar[d]\ar[r]α*&A\ar[d]g&\ar@[dr]|=&&B\ar[d]f\ar[r]α*&A\ar[d]\ar[r]1A*&A\ar[d]gX\ar[r]β&Y\ar[r]1Y&Y&&&X\ar[r]1X&X\ar[r]β&Y

Therefore, hom¯⁢(α*,β)=hom¯⁢(α*,1Y)∘hom¯⁢(1B*,β)=hom¯⁢(1A*,β)∘hom¯⁢(α*,1X), and hom¯ is a covariant functor, or that hom is a bifunctor.
