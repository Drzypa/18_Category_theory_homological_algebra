generator of a category


Let 𝒞 be a category, and f,g:A→B a pair of distinct morphisms. A morphism h:X→A is said to distinguish or separate f and g if f∘h≠g∘h. For example, if f≠g:A→B, then 1A on A distinguishes f and g.

A set S={Xi∣i∈I} of objects (indexed by a set I) is called a generating set of 𝒞 if any pair of distinct morphisms f,g:A→B can be distinguished by a morphism with domain in S and codomain A. In other words, there is h:Xi→A for some i∈I, such that f∘h≠g∘h. If {X} is a generating family of 𝒞, then X is called a generator of 𝒞. Any set of morphisms containing a generator is a generating set.

Examples

  1. 1.

    In Set, the category of sets, any singleton is a generator. Suppose f,g:A→B are distinct functions, so that f⁢(x)≠g⁢(x) for some x∈A. Let {y} be any singleton. Then h:{y}→A defined by h⁢(y)=x is the function distinguishing f and g: for f∘h⁢(y)=f⁢(x)≠g⁢(x)=g∘h⁢(y).

  2. 2.

    In Rng, the category of rings, the ring ℤ is a generator. If f,g:R→S are distinct ring homomorphisms, say, f⁢(r)≠g⁢(r) for some r∈R. Then the ring homomorphism h:ℤ→R given by h⁢(1)=r distinguishes f and g.

Remark. A projective object that is also a generator is called a progenerator.
