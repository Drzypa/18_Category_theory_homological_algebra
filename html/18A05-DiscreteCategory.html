discrete category


A category 𝒞 is said to be a discrete category if the only morphisms in 𝒞 are the identity morphisms associated with each of the objects in 𝒞.

For example, every set can be regarded as a discrete category. The objects are just the elements of the set. Furthermore, hom⁡(a,a) is identified with {a}, and hom⁡(a,b)=∅ if a≠b.

Remarks.

  • •

    A discrete category with one object is called a trivial category. For every category 𝒞, there is only one functor from 𝒞 to a trivial category. Hence, any trivial category is a terminal object in Cat, the category of small categories.

  • •

    A discrete category with no objects is called the empty category. For every category 𝒞, there is only one functor from the empty category to 𝒞. This functor is called the empty functor, where both the object and morphism functions are the empty set ∅. Thus, the empty category is the initial object in Cat.

  • •

    Given any category 𝒞, the smallest subcategory consisting of all objects in 𝒞 is discrete, which is also the largest discrete subcategory in 𝒞 (largest in the sense that it contains all objects of 𝒞). For every object X∈𝒞, we can associate the trivial category 𝒞X consisting of one object, X, and one morphism 1X.
