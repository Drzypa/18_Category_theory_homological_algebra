functor category


Let 𝒞,𝒟 be categories. Consider the class O of all covariant functors T:𝒞→𝒟, and the class M of all natural transformations τ:S⁢→˙⁢T for every pair S,T:𝒞→𝒟 of functors. Write 𝒟𝒞 for the pair (O,M).

For each pair of functors S,T:𝒞→𝒟, write hom⁡(S,T) the class of all natural transformations from S to T. If τ is in both hom⁡(S,T) and hom⁡(U,V), then S=U and T=V.

Using the composition of natural transformations, we have a mapping

∙:hom(R,S)×hom(S,T)→hom(R,T),

for every triple R,S,T:𝒞→𝒟. Since composition of natural transformations is associative, the associativity of ∙ applies.

In addition, for each S:𝒞→𝒟, we have the identity natural transformation 1S∈hom⁡(S,S). For every τ∈hom⁡(S,T) and every η∈hom⁡(T,S), we have τ∙1S=τ and 1S∙η=η.

From the discussion above, we are ready to call 𝒟𝒞 a category. However, unless hom⁡(S,T) is a set for every pair of functors in O, 𝒟𝒞 is not a category. When 𝒟𝒞 is a category, we call it the category of functors from 𝒞 to 𝒟, or simply a functor category.

That 𝒟𝒞 is a functor category depends on various restrictions being placed on the “sizes” of 𝒞 and 𝒟:

Proposition 1.

If C is U-small (http://planetmath.org/SmallCategory), then DC is a category.

Proof.

Suppose 𝒞 is 𝒰-small. Consider the class hom⁡(S,T). Each τ∈hom⁡(S,T) is determined by the collection of morphisms S⁢(A)→T⁢(A) for each object A in 𝒞. This means that, for each A in 𝒞, hom⁡(S⁢(A),T⁢(A)) contains the image of every τ∈hom⁡(S,T) under A. So the class of all these natural transformations is a subclass of the product

∏A∈Ob⁡(𝒞)hom⁡(S⁢(A),T⁢(A)) (1)

Since Ob⁡(𝒞), as well as each hom⁡(S⁢(A),T⁢(A)) is a set, so is the product (1). Hence hom⁡(S,T), being a subclass of (1), is a set, or that 𝒟𝒞 is a category. ∎

Proposition 2.

If in addition D is a U-category (http://planetmath.org/SmallCategory), then so is DC.

Proof.

𝒟 being a 𝒰-category means that hom⁡(S⁢(A),T⁢(A)) is 𝒰-small, for every object A in 𝒞. Since Ob⁡(𝒞) is also 𝒰-small (assumption in Proposition 1), the product (1) above is 𝒰-small. Consequently, hom⁡(S,T), being a subclass of (1), is 𝒰-small. This shows that 𝒟𝒞 is a 𝒰-category. ∎

Proposition 3.

If D is furthermore U-small, so is DC.

Proof.

We want to show that the class ℳ of all functors from 𝒞 to 𝒟 is 𝒰-small. A functor S:𝒞→𝒟 can be broken up into two components: a function S1:Ob⁡(𝒞)→Ob⁡(𝒟), and a function S2:Mor⁡(𝒞)→Mor⁡(𝒟), so that S2(A→B)=S1(A)→S1(B).

Define a binary relation ∼ on ℳ so that S∼T iff they have the same first component: S1=T1. It is easy to see that ∼ is an equivalence relation on ℳ. Let [S] be the equivalence class containing the functor S. For every morphism A→B, its image under the second component of every functor in [S] lies in hom⁡(S1⁢(A),S1⁢(B)). So the size of [S] can not exceed the size of

∏A,B∈Ob⁡(𝒞)hom⁡(S1⁢(A),S1⁢(B))

Since Ob⁡(𝒞) is 𝒰-small (assumption in Prop 1), so is Ob⁡(𝒞)×Ob⁡(𝒞). Furthermore, since each hom⁡(S1⁢(A),S1⁢(B)) is 𝒰-small (assumption in Prop 2), [S] is 𝒰-small as well.

Next, let us estimate the size of the class ℳ/∼ of equivalence classes in ℳ. First, note that for every functor S:𝒞→𝒟, its first component is a function from the set Ob⁡(𝒞) to the set Ob⁡(𝒟) by assumption. As [S]≠[T] iff S1≠T1, the size can not exceed

|Ob(𝒟)Ob⁡(𝒞)|

the cardinality of the set of all functions from Ob⁡(𝒞) to Ob⁡(𝒟). By assumption, Ob⁡(𝒟) is 𝒰-small, so is Ob(𝒟)Ob⁡(𝒞). As a result, ℳ/∼ is 𝒰-small. Together with the fact that [S] is 𝒰-small for each functor S, we have that ℳ itself must be 𝒰-small, which completes the proof. ∎
