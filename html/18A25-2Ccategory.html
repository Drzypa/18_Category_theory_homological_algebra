2-C*-category


Definition 0.1.

A 2-C* -category, 𝒞*2, is defined as a (small) 2-category for which the following conditions hold:

  1. 1.

    for each pair of 1-arrows (ρ,σ) the space H⁢o⁢m⁢(ρ,σ) is a complex Banach space.

  2. 2.

    there is an anti-linear involution ‘*’ acting on 2-arrows, that is,

    *:Hom(ρ,σ)→Hom(ρ,σ),

    (S↦S* ) with ρ and σ being 2-arrows;

  3. 3.

    the Banach norm is sub-multiplicative (that is,

    ∥T∘S∥≤∥S∥⁢∥T∥,

    when the composition is defined, and satisfies the C* -condition:

    ∥S*∘S∥=∥S2∥;
  4. 4.

    for any 2-arrow S∈H⁢o⁢m⁢(ρ,σ), S*∘S is a positive element in H⁢o⁢m⁢(ρ,ρ), that is often denoted as E⁢n⁢d⁢(ρ).

Remark 0.1.

With the above notations, the set of 2-arrows E⁢n⁢d⁢(ι⁢A) is a commutative monoid, with the identity map ι:𝒞*20→𝒞*21 assigning to each object A∈𝒞*20 a 1-arrow ι⁢A such that:

s⁢(ι⁢A)=t⁢(ι⁢A)=A.
