types of morphisms


Definition 1.

A morphism f:A→B is a monomorphism, if for any two morphisms g,h:C→A the equality f∘g=f∘h implies h=g.

Dual (http://planetmath.org/DualityPrinciple) notion: Morphism f:A→B is an epimorphism, if for any two morphisms g,h:B→C the equality g∘f=h∘f implies h=g.

A morphism f is a bimorphism, if it is monomorphism and epimorphism at the same time. Also the names epimorphic extension and epimorphic monomorphism are used.

Definition 2.

A morphism f:A→B is called retraction if there exists a morphism g:B→A such that f∘g=i⁢dB.

Retractions are sometimes called split epimorphisms.

Dual notion: a morphism f:A→B is a section (or coretraction or split monomorphism) if there exists a morphism g:B→A such that g∘f=i⁢dA.

A morphism f:A→B is an isomorphism (http://planetmath.org/Isomorphism2) if it is a retraction and section at the same time.

Bimorphism and isomorphism are examples of self-dual properties. The condition that f is isomorphism is equivalent to the existence of a morphism g with f∘g=i⁢dB and g∘f=i⁢dA (for the proof see properties of monomorphisms and epimorphisms).

Definition 3.

If f is an isomorphism then the morphism g:B→A such that f∘g=i⁢dB and g∘f=i⁢dA is called inverse morphism of f and denoted by f-1.

Definition 4.

If there exists an isomorphism f:A→B we say that the objects A and B are isomorphic, denoted by A≅B.
