a functor is an equivalence iff it is fully faithful and essentially surjective


Proposition 1.

A functor is an equivalence iff it is full, faithful and essentially surjective.

Proof.

(⇒). Let functor F:C→D be an equivalence. So there is a functor G:D→C such that F⁢G≊1D and G⁢F≊1C, where ≊ means naturally isomorphic. For F⁢G≊1D, this means that there are natural transformations

α :F⁢G⇒1D
α* :1D⇒F⁢G

such that for any x¯∈𝒪⁢ℬ⁢(D), αx¯⁢αx¯*=1x¯ and αx¯*⁢αx¯=1F⁢G⁢(x¯), meaning that x¯ and F⁢G⁢(x¯) are isomorphic (as objects of D). Similarly, for G⁢F≊1C, there are natural transformations

β :G⁢F⇒1C
β* :1C⇒G⁢F

such that for any x∈𝒪⁢ℬ⁢(C), βx⁢βx*=1x and βx*⁢βx=1G⁢F⁢(x), meaning that x and G⁢F⁢(x) are isomorphic (as objects of C).

For any x¯∈𝒪⁢ℬ⁢(D), let x=G⁢(x¯). To show that F⁢(x)≅x¯ is the same as showing F⁢G⁢(x¯)≅x¯. By the previous discussion, this is clear. Therefore, F is essentially surjective.

We now show that F is faithful. Let f∈hom⁡(x,y) for x,y∈𝒪⁢ℬ⁢(C). Then by the above, there is a commutative diagram

\xymatrix⁢@⁢C=1.5⁢c⁢m⁢x⁢\ar⁢[r]f⁢\ar⁢[d]βx*⁢&⁢y⁢\ar⁢[d]βy*⁢G⁢F⁢(x)⁢\ar⁢[r]G⁢F⁢(f)⁢&⁢G⁢F⁢(y)

Thus βy*⁢f=G⁢F⁢(f)⁢βx*; applying βy to both sides we have f=βy⁢G⁢F⁢(f)⁢βx*. Thus f is determined by G⁢F⁢(f) so that G⁢F must be faithful. Applying the same argument to a morphism g∈hom⁡(x¯,y¯) for x¯,y¯∈𝒪⁢ℬ⁢(D) and using F⁢G in place of G⁢F shows that G is faithful as well.

Next we show that F is full. Suppose g∈hom⁡(F⁢(x),F⁢(y)) for x,y∈𝒪⁢ℬ⁢(C). Let f=βy⁢G⁢(g)⁢βx*. Then the following diagram commutes for either choice of horizontal arrow on the bottom: the top arrow by definition of β and β*; the bottom arrow by definition of f.

\xymatrix⁢@⁢C=1.5⁢c⁢m⁢x⁢\ar⁢[r]f⁢\ar⁢[d]βx*⁢&⁢y⁢\ar⁢[d]βy*⁢G⁢F⁢(x)⁢\ar⁢@⁢<.5⁢e⁢x>⁢[r]G⁢F⁢(f)⁢\ar⁢@⁢<-.5⁢e⁢x>⁢[r]G⁢(g)⁢&⁢G⁢F⁢(y)

It follows that G⁢(g)⁢βx*=βy*⁢f=G⁢F⁢(f)⁢βx*; composing on the right with βx gives G⁢(g)=G⁢F⁢(f). Finally, since G is faithful, we have g=F⁢(f) and F is full.

(⇐). Now, we assume that F:C→D is essentially surjective, full and faithful. We want to show that F is an equivalence.

Since F is fully faithful, for any x,y∈𝒪⁢ℬ⁢(C), the hom-sets hom⁡(x,y) and hom⁡(F⁢(x),F⁢(y)) are bijective. In particular, since hom⁡(x,x)≅hom⁡(F⁢(x),F⁢(x)), 1x gets bijectively mapped to 1F⁢(x).

Next, let a,b∈𝒪⁢ℬ⁢(D), we can find x,y∈𝒪⁢ℬ⁢(C) such that F⁢(x)≅a (via a bijective morphism, say α:F⁢(x)→a) and F⁢(y)≅b (via bijection β:F⁢(y)→b). So, given g:a→b, we have β-1⁢g⁢α:F⁢(x)→F⁢(y). Similarly, for f:F⁢(x)→F⁢(y), we have β⁢f⁢α-1:a→b. Therefore, hom⁡(a,b)≅hom⁡(F⁢(x),F⁢(y))≅hom⁡(x,y). In particular, for any a∈𝒪⁢ℬ⁢(D) with x∈𝒪⁢ℬ⁢(C) such that F⁢(x)≅a, since hom⁡(a,a)≅hom⁡(x,x), 1a gets mapped bijectively to 1x.

If, for any a∈𝒪⁢ℬ⁢(D), there are x1,x2∈𝒪⁢ℬ⁢(C) such that F⁢(x1)≅a≅F⁢(x2), then hom⁡(x1,x2)≅hom⁡(a,a)≅hom⁡(x2,x1). Suppose 1a gets mapped (bijectively) to s:x1→x2 and to t:x2→x1. Because hom⁡(a,a)≅hom⁡(a,a)×hom⁡(a,a)≅hom⁡(x1,x2)×hom⁡(x2,x1), 1a gets mapped to s⁢t:x1→x1. But 1a is also mapped bijectively to 1x1:x1→x1, this shows that s⁢t=1x1. Similarly, t⁢s=1x2. So, we have x1≅x2.

Define [x]={y∣y≅x} and pick a representative x0 with a bijection x→x0. Any a∈𝒪⁢ℬ⁢(D) can be mapped bijective into members of [x] can be mapped bijectively to x0. So, for any a∈[a]⊆𝒪⁢ℬ⁢(D), we have a well-defined G⁢(a):=x0.

For any morphism g:a→b, a,b∈𝒪⁢ℬ⁢(D), pick representatives a0∈[a] and b0∈[b], then there is a unique morphism g*:a0→b0 given by \xymatrix⁢@⁢C=0.5⁢c⁢m⁢a0⁢\ar⁢[r]⁢&⁢a⁢\ar⁢[r]g⁢&⁢b⁢\ar⁢[r]⁢&⁢b0. Since hom⁡(a,b)≅hom⁡(a0,b0)≅hom⁡(x0,y0), g* in turns gets mapped to a unique morphism in hom⁡(x0,y0). We call this G⁢(g).

Now, we are ready to define the functor G:D→C. For an object a, we define G⁢(a) to be the unique x0∈𝒪⁢ℬ⁢(C) discussed two paragraphs ago. For a morphism g:a→b, we define G⁢(g) to be the unique morphism f:G⁢(a)→G⁢(b) that is found in the previous paragraph. In order for this to be a functor, we need to check the following:

  • •

    G⁢(1a)=1G⁢(a), and

  • •

    G⁢(g1⁢g2)=G⁢(g1)⁢G⁢(g2).

These can be easily verified, as follows: 1a is mapped to

\xymatrix⁢@⁢C=1⁢c⁢m⁢a0⁢\ar⁢[r]⁢&⁢a⁢\ar⁢[r]1a⁢&⁢a⁢\ar⁢[r]⁢&⁢a0=a0⁢\ar⁢[r]-⁢1a0⁢&⁢a0,

and 1a0 is mapped to 1x0=1G⁢(a). The second property can be verified similarly.

Finally, we want to show that F⁢G≊1D and G⁢F≊1C. This amounts to showing that for each x∈𝒪⁢ℬ⁢(C), G⁢F⁢(x)≅x, and for each a∈𝒪⁢ℬ⁢(D), F⁢G⁢(a)≅a. For the first part, x gets mapped to F⁢(x) which is in turn mapped to G⁢F⁢(x)=x0, where F⁢(x0)≅F⁢(x), which means that x0≅x, or that G⁢F⁢(x)≅x. For the second part, a is mapped first to G⁢(a)=x0, where F⁢(x0)≅a. Then x0 is in turn mapped to F⁢(x0). Thus, F⁢G⁢(a)=F⁢(x0)≅a. ∎
