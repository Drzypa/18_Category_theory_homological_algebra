product of categories


There are occasions when we need to consider objects from different categories being paired up. For example, if 𝒞 is a category, then hom⁡(A,B) where A,B are objects of 𝒞 is a set, and we can think of hom as a functor from 𝒞op×𝒞 to the category of sets. But what is 𝒞op×𝒞 exactly? We will give this a formal definition presently.

Let 𝒞 and 𝒟 be categories. Define the Cartesian product 𝒞×𝒟 of 𝒞 and 𝒟 as the following pair (O,M), where

  • •

    O is the class consisting of ordered pairs (X,Y), where X is an object in 𝒞 and Y is an object in 𝒟

  • •

    M is the class consisting of ordered pairs (f,g), where f is a morphism in 𝒞 and g is a morphism in 𝒟.

There is a category structure on 𝒞×𝒟. But several things need to be defined first.

  1. 1.

    Elements of O are called the objects of 𝒞×𝒟 and elements of M are called the morphisms of 𝒞×𝒟. For each morphism (f,g)∈M, we define the domain and codomain operations

    dom⁡(f,g):=(dom⁡(f),dom⁡(g))  and  cod⁡(f,g):=(cod⁡(f),cod⁡(g)).

    Note that for simplicity, we have used the same symbol dom and cod for 𝒞,𝒟, and 𝒞×𝒟.

  2. 2.

    Next, for each pair of objects A,B∈𝒞×𝒟, we have a set hom⁡(A,B) consisting of all morphisms in 𝒞×𝒟 whose domain is A and codomain is B. Note that hom⁡(A,B) is a set because it is hom⁡(X,Z)×hom⁡(Y,T), where A=(X,Y) and B=(Z,T) and each component in the product is assumed to be a set.

  3. 3.

    Finally, for objects A,B,C in 𝒞×𝒟, we have a function ∘ called composition:

    ∘:hom(A,B)×hom(B,C)→hom(A,C).

    To define ∘, write each object A,B,C as ordered pairs: A=(X,Y), B=(Z,T), C=(U,V). In addition, let α=(f,g)∈hom⁡(A,B) and β=(p,q)∈hom⁡(B,C). Then

    ∘(α,β):=(∘1(f,p),∘2(g,q)),

    where ∘1 and ∘2 are compositions defined in 𝒞 and 𝒟 respectively, such that

    ∘1:hom(X,Z)×hom(Z,U)→hom(X,U) and ∘2:hom(Y,T)×hom(T,V)→hom(Y,V).

    As usual, we write β∘α for ∘(α,β).

  4. 4.

    Now, it is not hard to see that 𝒞×𝒟 with ∘ is a category. For example, let us verify that (A,B)≠(C,D) implies hom⁡(A,B)∩hom⁡(C,D)=∅. Write A=(X,Y), B=(Z,S), C=(T,U) and D=(V,W). Suppose α=(f,g)∈hom⁡(A,B)∩hom⁡(C,D). Then f∈hom⁡(X,Z)∩hom⁡(T,V) and g∈hom⁡(Y,S)∩hom⁡(U,W). But this implies X=T, Z=V, Y=U, and S=W. So A=(X,Y)=(T,U)=C and B=(Z,S)=(V,W)=D.

Remarks.

  • •

    The above construction can be generalized to n-fold Cartesian products. If 𝒞1,…,𝒞n be categories. Then 𝒞:=𝒞1×⋯⁢𝒞n can be defined much the same way as in the case n=2. 𝒞 is a category and is sometimes written ∏i=1n𝒞i.

  • •

    Associated with this product, we can form n (covariant) functors called projection functors Πi:𝒞→𝒞i, given by Πi⁢(A)=Ai and Πi⁢(α)=αi, where A=(A1,…,An) and α=(α1,…,αn).

  • •

    The product 𝒞 of 𝒞i also enjoys the universal property that for every category 𝒟 and functors Fi:𝒟→𝒞i, there is a unique functor F:𝒟→𝒞 such that Πi∘F=Fi (in other words, Fi factors through F).

  • •

    If fact, any category that enjoys the universal property described above is naturally equivalent to the product of 𝒞i. We may actually define product category this way, and then prove its existence using the construction that is given as the definition at the beginning of this article.

  • •

    More generally, we can define arbitrary (direct) product of categories. The definition is completely similar to the one above. If {𝒞i∣i∈I} is a family of categories indexed by a set I, we often write ∏i∈I𝒞i as the product category. Objects and morphisms are written (Ai)i∈I and (αi)i∈I respectively. When all the 𝒞i are identical, say, equal to 𝒞, we also write the product as 𝒞I, and call it the I-fold direct product of 𝒞.

  • •

    The existence of the product of categories indexed by an arbitrary set shows that the category of (small) categories Cat has products.

  • •

    Let 𝒞=𝒟×ℰ. Then we may identify 𝒟 as a subcategory of 𝒞: for each object E in ℰ, define FE:𝒟→𝒞, by FE⁢(A):=(A,E) and F⁢(α)=(α,1E). Then FE is a faithful functor. The image 𝒞E of FE (with objects (A,E) and morphisms (α,1E)) is a subcategory of 𝒞. It is not hard to see that 𝒟 and 𝒞E are isomorphic as categories (http://planetmath.org/CategoryIsomorphism).

  • •

    The above also shows that for any objects A,B in ℰ, 𝒞A and 𝒞B are isomorphic.
