A subcategory $\Cat A$ of a category $\Cat B$ is said to be \emph{isomorphism-closed} if for
any $A\in\Cat A$ and a $\Cat B$-isomorphism $\Map hAB$, also the $\Cat B$-object $B$ belongs
to $\Cat A$. 

More simply: the subcategory $\Cat A$ contains with each object all isomorphic $\Cat
B$-objects.

Another name commonly used for isomorphism-closed subcategories is \emph{replete subcategory}.

This condition is very natural. E.g~in the category of topological spaces we usually study
properties which are invariant under homeomorphisms -- so called topological properties.
Every topological property corresponds to a strictly full subcategory of $\Top$. 

A subcategory which is isomorphism-closed and full is called \emph{strictly full}.
